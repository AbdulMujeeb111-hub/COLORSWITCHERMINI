# Color Switcher by Abdilmujeeb under NOORLABS MENTORSHIP

This is a simple mini project that allows users to change the background color of display tiles/areas on this webpage.

## Description
This project demonstrates basic JavaScript features such as events,functions, DOM manipulation, etc. Each tile was assigned a blue background color on default and when the select dropdown is used the selected tile color changes to the chosen color.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Change background color of the tile by choosing preferred color out of options provided.
- Simple and beginner-friendly design
- Current color being displayed is shown on the tile. Can be useful for you if you like that exact color and want to use it in a design of yours later.

## How It Works
1. Tiles are created with blue background colors from the start.
2. JavaScript listens for button click events via the event delegation feature.
3. On click, the current tile clicked on   changes background color to a new random color whose name will then be outputed.

## How to Run
1. Download or clone the repository.
2. Open `index.html` in your browser.
3. The click the select dropdown as you deem fit and watch the GOAT JavaScript run it's magic.
